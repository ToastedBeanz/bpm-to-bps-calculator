# BPM to BPS Calculator

![GitHub code size in bytes](https://img.shields.io/github/languages/code-size/ToastedBeanz/BPM-to-BPS-calculator)


this program will take your input in BPM (Tempo/Beats-Per-Minute), and convert it to BPS (Beats-Per-Second). It will also calculate the time between every Beat (in seconds)
